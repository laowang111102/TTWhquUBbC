## 技术栈 java260

- 前端框架：Vue3
- UI组件库：Element Plus
- 构建工具：Vite
- 状态管理：Pinia
- 路由管理：Vue Router
- 数据可视化：ECharts

## 功能模块

系统包含以下12个页面：

- 登录页
- 注册页
- 仪表盘首页
- 水果分类管理
- 水果库存管理
- 订单管理
- 用户管理
- 角色管理
- 权限管理
- 个人中心
- 系统设置
- 错误页面

### 自定义组件

共包含15个自定义组件。

### 功能特点

- 分页功能
- Pinia实现状态管理
- 数据图表与ECharts可视化
- 父子组件通信（prop与emit）
- Vue指令支持（v-if、v-for、v-on、v-model、v-bind）
- 计算属性与侦听器
- RBAC权限控制系统

## 系统角色

- 系统管理员：负责系统管理、用户管理、角色权限分配等操作。
- 普通用户：负责日常业务操作，如查看库存、管理订单等。

## 运行环境

- Node.js环境（推荐使用LTS版本）
- 现代浏览器（Chrome、Firefox、Safari等）

## 部署步骤

1. 克隆项目到本地
2. 安装Node.js环境
3. 安装项目依赖
4. 本地运行与调试
5. 构建生产环境代码
6. 部署到服务器
